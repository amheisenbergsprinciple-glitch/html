08   CHALLENGES

Known Risks. Known Fixes.

[R1]

API RATE LIMITS

HIBP and third-party APIs impose usage caps that may restrict scan volume at scale.

// MITIGATION

Implement intelligent caching, request queuing, and tiered API usage plans to stay within limits while serving users.

[R2]

DATA ACCURACY

Breach databases may have outdated, incomplete, or false-positive information affecting results.

// MITIGATION

Cross-verify across multiple independent sources. Display confidence scores alongside results for full transparency.

[R3]

PRIVACY & TRUST

Users may be uncomfortable submitting personal data to a third-party security tool.

// MITIGATION

Zero data retention policy. HTTPS encryption at all times. Fully transparent, open-source privacy policy published on launch.
